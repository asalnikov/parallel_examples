Несколько слов o том, что лежит в этой директории.

.bash_profile - образец файла, с подгружаемыми модулями, чтобы положить
                в домашний каталог на ломоносове

config.example - пример конфига для openssh. отредактировать по своему вкусу и 
                 дополнить имеющийся ${HOME}/.ssh/config (если имеется) текстом из
                 данного файла.


Makefile и исходник - скомпилировать при помощи программы make и 
                      получившиеся бинарники положить в каталог
                      ${HOME}/_scratch.
